# 8-Puzzle Game (C++)

## Description
This project is a console-based implementation of the classic 8-Puzzle game.
The goal is to arrange the tiles in the correct order (as shown at the start of the game) by sliding them into the empty space.

## Features
- Console-based gameplay
- User input for moves
- Win Condition detection

## How to Run
1. Open the project folder in a C++ compiler or IDE.
2. Compile the source file.
3. Run the executable and play the game in the console.
